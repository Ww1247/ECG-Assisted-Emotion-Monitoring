# 硬件构建手册

这个部分主要记录硬件的搭建过程，以及在搭建过程中可能遇到的问题和解决方法。

## 传感器

本项目的传感器如下表所示：

| 项目  | 传感器名称         | 型号    | 描述                         | 数量 (pcs)     | 规格书      |
|:----:|------------------|:--------:|-------------------------------|:---------:|:-----------:|
|  1   | 摄像头           | WebCam   | 采集面部图像和照片              | 1         |             |
|  2   | 脉搏传感器       | MAX30102 | 采集ECG心电图、脉搏、血氧数据    | 1         |             |
|  3   | 湿度、温度传感器 | AHT20    | 采集手指的湿度和温度变化         | 1         |             |
